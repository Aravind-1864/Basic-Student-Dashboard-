# 🎓 Student Dashboard

A simple and responsive Student Dashboard built using HTML and CSS.

## Features

- 📌 Sidebar Navigation
- 👋 Welcome Section
- 📊 Student Statistics Cards
- 📅 Timetable Section
- 🔔 Notifications Section
- 📈 Attendance Progress Bar
- 🎨 Clean and Modern UI
- 📱 Responsive Design

## Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid
